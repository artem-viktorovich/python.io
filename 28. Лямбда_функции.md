## Лямбда функции

```python
lambda parametrs: expression
```

_**lambda**_ - ключевое слово
_**parametrs**_ - может быть несколько параметров
_**expression**_ - допускается только одно выражение

> Лямбда функции в Python — это небольшие анонимные функции, которые определяются с помощью ключевого слова lambda. Они позволяют создавать функции на лету, без необходимости в использовании стандартного определения функции с помощью def.

Разница между функцией и лямбда функцией

```python
def mult(a, b):  
    return a * b
```

```python
lambda a, b: a * b
```

> Важно помнить, что может сработать форматирование и формула редактируется в Vs Code так как по документации PEP 8 лямбда функцию не стоит применять
> PyCharm не изменяет формулу

Пример использование функции лямбда

```python
mult = lambda a, b: a * b  
print(mult(2, 3))  # 6
```

```python
numbers = [1, 2, 3, 4]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # Вывод: [1, 4, 9, 16]
```

- В этом примере у нас есть список numbers, содержащий числа от 1 до 4.
- Функция map применяется к списку numbers, и для каждого элемента этого списка вызывается лямбда функция lambda x: x ** 2.
- Здесь x — это каждый элемент списка, а выражение x ** 2 возводит этот элемент в квадрат.
- Функция map возвращает итератор, содержащий результаты применения лямбда функции, а затем мы преобразуем этот итератор в список с помощью list(). В итоге получаем новый список, содержащий квадраты чисел: [1, 4, 9, 16].

```python
numbers = [1, 2, 3, 4, 5, 6]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # Вывод: [2, 4, 6]
```

- В этом примере у нас есть список numbers, содержащий числа от 1 до 6.
- Функция filter принимает лямбда функцию и список numbers. Лямбда функция lambda x: x % 2 == 0 проверяет, четное ли число (остаток от деления на 2 равен 0).
- filter проходит по каждому элементу списка и отбирает только те значения, для которых лямбда функция возвращает True.
- Как и в предыдущем примере, мы используем list() для преобразования результата в список. Получаем новый список, содержащий только четные числа: [2, 4, 6].

```python
def gretting(greet):  
    return lambda name: f"{greet}, {name}!"  
  
  
morning_gretting = gretting("Доброе утро")  
print(morning_gretting('Артем'))  # Доброе утро, Артем!
  
evening_greeting = gretting("Доброе вечер")  
print(evening_greeting('Артем'))  # Доброе вечер, Артем!
```
