### –ú–µ—Ç–æ–¥—ã –∫–æ—Ä—Ç–µ–∂–µ–π

>–ú–µ—Ç–æ–¥—ã –∫–æ—Ä—Ç–µ–∂–µ–π –Ω–∞—Å–ª–µ–¥—É–µ—Ç—Å—è –æ—Ç –∫–ª–∞—Å—Å–∞ tuple


```python
# index()
my_id = (456, 46, 789, 234, 56, 56)  
  
print(my_id.index(56))   #1 –µ—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç –≤—Å—Ç—Ä–µ—á–∞–µ—Ç—Å—è –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ä–∞–∑, —Ç–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è –∏–Ω–¥–µ–∫—Å –ø–µ—Ä–≤–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞
```

>–ï—Å–ª–∏ –Ω—É–∂–Ω–æ –∏–∑–º–µ–Ω–∏—Ç—å –∫–æ—Ä—Ç–µ–∂, —Ç–æ –∫–æ–Ω–≤–µ—Ä—Ç–∏—Ä—É–µ–º –µ–≥–æ –≤ —Å–ø–∏—Å–æ–∫, –∏–∑–º–µ–Ω—è–µ–º –∏ –∫–æ–Ω–≤–µ—Ä—Ç–∏—Ä—É–µ–º –æ–±—Ä–∞—Ç–Ω–æ –≤ –∫–æ—Ä—Ç–µ–∂

```python
posts_id = (345, 567)  
  
posts_id_list = list(posts_id)  
posts_id_list.append(34)  
print(type(posts_id_list))   #<class 'list'>
print(posts_id_list)   #[345, 567, 34]
  
posts_id_tuple = tuple(posts_id_list)  
print(type(posts_id_tuple))   #<class 'tuple'>
print(posts_id_tuple)   #(345, 567, 34)
```

–°—É—â–µ—Å—Ç–≤—É–µ—Ç —Å–ø–æ—Å–æ–± —É–∑–Ω–∞—Ç—å –∏–Ω–¥–µ–∫—Å—ã –ø–æ–≤—Ç–æ—Ä—è—é—â–∏—Ö—Å—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –∫–æ—Ä—Ç–µ–∂–µ

```python

```
